About ROT13 Cipher:

ROT13 is a special case of the Caesar cipher, which was developed in ancient Rome. The ROT13 cipher shifts each letter by 13 places in the alphabet. This means that 'A' becomes 'N', 'B' becomes 'O', and so on. The cipher is symmetrical, so applying it twice brings you back to the original text.

How to Decode Using ROT13 Cipher:

  • Identify the encoded message from stage seven.
  • Apply the ROT13 transformation to each letter in the message.
  • Convert the resulting letters to obtain the decoded message.
  • Enter the decoded message in the input field below and submit to proceed.

Example:

Let's decode the message “URYYB” using the ROT13 cipher:

  • U (ROT13 → H)
  • R (ROT13 → E)
  • Y (ROT13 → L)
  • Y (ROT13 → L)
  • B (ROT13 → O)

The decoded message is: HELLO
