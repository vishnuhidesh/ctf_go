Decode the text obtained from stage two using the Caesar Cipher with a key of 7:

About Caesar Cipher:

The Caesar cipher is a type of substitution cipher where each letter in the plaintext is shifted a certain number of places down or up the alphabet. In this challenge, you need to decode the message by shifting each letter backwards by the key value.

Read more about Caesar Cipher here.

Steps to Decode Using Caesar Cipher:

  1. Identify the Caesar cipher key provided. In this case, the key is 7.
  2. Shift each letter in the given message backwards by 7 positions in the alphabet. For example, if you encounter the letter 'H', shifting it backwards by 7 positions would give you 'A'.
  3. Apply this backward shift to all letters in the message to reveal the decoded text.
  4. Enter the decoded message in the input field below and submit to proceed to the next stage.

Example:

Let's decode the message "MJQQT" using the Caesar cipher with a key of 5.

  1. Identify the key, which is 5.
  2. Shift each letter in "MJQQT" backwards by 5 positions:
Original:  M  J  Q  Q  T
Shifted:   H  E  L  L  O
  1. The decoded message is: "HELLO"
